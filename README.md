# Weather Monitoring System
A Weather Monitoring System is an electronic setup designed to measure, record, and display various environmental parameters such as temperature, humidity, atmospheric pressure, and rainfall. These systems are commonly used in meteorology, agriculture, and climate research.

This project is based on Arduino Uno, which collects real-time data from various sensors (DHT11, BMP180, and Rain Sensor) and displays it on an LCD screen. It can be extended to store data in a database or send it to a cloud server for remote monitoring.
